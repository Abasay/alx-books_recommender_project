
# BOOKS WEBSITE

## Installation

Installing this project :

```bash
  git clone https://github.com/Abasay/alx-books_website_project

  cd alx-books_website_project

```
#initialize by:
```bash
  npm install
```

#start project by:
```bash
  npm start
```
